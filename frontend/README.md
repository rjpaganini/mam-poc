## Key Information
- Development server runs on port 3001 (not the default 3000)
- WebSocket connects to backend on ws://localhost:5001/ws
- Electron app wraps the React frontend

## Getting Started
```bash
cd frontend
npm install
npm start  # Runs on port 3001
```
